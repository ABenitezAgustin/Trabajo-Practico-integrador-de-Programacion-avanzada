Sistema de Gestión de Biblioteca Digital

Descripción

El Sistema de Gestión de Biblioteca Digital es una aplicación desarrollada en Python como parte del Trabajo Práctico Final de la materia Programación Avanzada.
El objetivo del proyecto es facilitar la administración de una biblioteca mediante un sistema simple e intuitivo que funciona a través de un menú por consola. Desde el programa es posible realizar el alta, baja, modificación y listado de libros y usuarios, además de registrar préstamos y devoluciones de libros, verificando que un mismo ejemplar no pueda ser prestado más de una vez al mismo tiempo.
Durante el desarrollo del proyecto se aplicaron los principales conceptos de Programación Orientada a Objetos, como la creación de clases y objetos, herencia, composición, agregación y polimorfismo, con el objetivo de mantener un código organizado y fácil de entender. El sistema fue diseñado de manera modular, separando la gestión de libros, usuarios y préstamos para que cada parte funcione de forma independiente y sea más sencilla de mantener.
Este trabajo nos permitió poner en práctica los conocimientos adquiridos durante la cursada, enfrentándonos a un caso de uso real y desarrollando una aplicación funcional que puede seguir ampliándose con nuevas características y mejoras.
